# 前置知识

libbpf-bootstrap 是一个脚手架操场，为初学者用户设置尽可能多的东西，让他们直接开始编写 BPF 程序并修改它们，而无需对初始设置造成不必要的挫折。

# 题目分析

直接用 ida 打开可以发现是一个直接套了 libbpf/libbpf-bootstrap 模版的程序，逻辑必然在 ebpf 程序中。

# 题解
